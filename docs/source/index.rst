Добро пожаловать в документацию CLS!
===================================

**CLS** - это Python-проект для автоматизированной работы загрузке, обработке, хранению данных
и использованию этих данных для тренировки ML моделей.

Перейдите в раздел :doc:`usage` для дальнейшей информации, 
включая как установить проект. 

.. note::

   This project is under active development.
