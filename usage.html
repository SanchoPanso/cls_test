Использование

Установка

Чтобы использовать CLS, сначала установите его следующим образом:

$ git clone https://github.com/SanchoPanso/CLS.git
$ cd CLS
$ python3 -m venv venv
$ source venv/bin/activate
(venv) $ pip install -r requirements.txt
(venv) $ pip install -e .

Модули

Проект разделен на модули, отвечающие за свою отдельную задачу.

  • Classification - модуль отвечает за тренировку моделей классификации. В его задачи входит скачивание изображений, нахождение сегментаций, тренировка с использованием W&B, инференс полученных моделей и др.

  • Instance - модель отвечает за скачивание датасета сегментации с roboflow и тренировку модели YOLO.

  • Demonstration - модуль отвечает за демонтрацию моделей с использованием gradio.
