PostgreSQL (Сегментация)¶

Для того, чтобы сегментационные маски были доступны для всех пользователей, создается отдельный контейнер с образом базы данных PostgreSQL.

Создание новой БД¶

  1. Создать контейнер:

cd postgres
docker compose up -d

2. Указать URL к БД в файле конфигурации по схеме: Пример:

url: ...
