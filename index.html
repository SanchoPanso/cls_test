Добро пожаловать в документацию CLS!

CLS - это Python-проект для автоматизированной работы загрузке, обработке, хранению данных и использованию этих данных для тренировки ML моделей.

Перейдите в раздел Использование для дальнейшей информации, включая как установить проект.

Примечание

This project is under active development.
